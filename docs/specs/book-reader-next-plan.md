## 1. 目标顺序

稳态推进，**一刀一验**，不并行开大坑。

```text
① 打开加速  →  ② 选区菜单 + 划线（v1 + 规范）  →  ③ 书内搜索
     →  ④ 目录/书签（+ 笔记段）  →  ⑤ 听书/图片/其余（按需）
```

选区气泡的**交互规矩与槽位表**以 [book-reader.md · 选区菜单与划线](./book-reader.md) 为准（**两段式**：① 动作条 → 点划线才出 ② 样式面板；贴边翻转）。本页只跟实施刀序。上行新功能 = 改槽位表 → 再接线。

| 优先级 | 主题 | 状态 |
