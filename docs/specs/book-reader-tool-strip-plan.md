## 1. 背景

当前图书 chrome（[`book_reader_chrome.dart`](../../lib/presentation/widgets/reader/book_reader_chrome.dart)）：

- **顶栏**：返回 · 书名 · 进度% · 目录 · 书签 · 排版  
- **底栏**：上一页 · 节标签 · 下一页  
- **排版**：单一 bottom sheet（字号 / 行距 / 边距 / 主题 / 模式）

目标对齐微信读书：点中间显隐 chrome 后，**底栏换成工具条**——上为进度、下为五键；点键在工具条**上方**展开对应面板。

```text
┌──────────────────────────────────┐
│  （展开区：亮度 / 排版 / 阅读模式）   │
├──────────────────────────────────┤
│   < ════════●═══════════════ >   │  进度条
├──────────────────────────────────┤
│  目录   听书   亮度   排版   模式  │  五键
└──────────────────────────────────┘
```

## 2. 目标交互（v1）

### 2.1 底栏

| 区域 | 行为 |
|------|------|
| 进度条 | 左右短箭头 = 上一页/下一页（翻页模式）或上一节/下一节（滚动）；轨道拖动 seek，值用全书 `progressFraction`（Foliate loc 同源），松手 `goToPercent` |
| 目录 | 打开现有 TOC drawer |
| 听书 | **占位**（disabled 或 SnackBar「即将推出」）；本大改不做 TTS |
| 亮度 | 展开亮度面板（见 2.3） |
| 字体排版 | 展开排版面板（迁现有 settings 内容） |
| 阅读模式 | 展开模式面板（滚动/翻页、翻页效果；桌面无滚动项） |

- 五键互斥：同一时刻最多展开一个面板；再点同一键收起。  
- 进度条与五键同属底栏玻璃层；面板叠在五键之上、仍在 chrome 内（非新 route）。

### 2.2 顶栏（精简）

- **保留**：返回、书名（可选一行小进度%）。  
- **挪走**：目录、排版 → 底键。  
- **书签**：v1 **暂留顶栏**（添加/列表），避免和目录面板二期纠缠；或标记为刀 G。

### 2.3 各面板内容（v1）

| 面板 | 纳入 | 不做 |
|------|------|------|
| 亮度 | 亮度滑杆（阅读内遮罩调暗，不改系统亮度） | 系统屏幕亮度 API、AUTO 传感器 |
| 字体排版 | 主题大卡（阅读背景）；字号滑杆 + **B**；行距滑杆；水平/垂直页边距；字体子面板；更多设置子面板 | 导入字体文件、补充字体库、系统字体深层列表 |
| 阅读模式 | 滚动/翻页（`BookReaderCapabilities`）、`pageTurnEffect` | 真仿真卷曲、左手模式、音量键翻页 |
| 目录 | 现有 TOC 列表跳转 | 笔记 Tab、微信式「目录/笔记/书签」三段（可二期） |

### 2.4 显隐与桌面

- 默认隐藏；点空白/中间显隐整套顶+底（含展开面板时一并收起）。  
- 桌面：顶栏继续让开红绿灯 / titleBar inset；底栏同构；宽屏双栏策略不变。  
- 漫画 chrome **本大改不改**（组件可先 book-only，命名勿绑死 comic）。

## 3. 分刀顺序

| 刀 | 内容 | 验收 |
|----|------|------|
| **A 壳** | 底栏换成「进度占位 + 五键」；顶栏去掉目录/排版；展开区空壳与选中态；更新本计划对应 specs 条目 | 显隐正确；键可点高亮 |
| **B 进度** | 真绑定 `progressFraction` + seek + 左右翻页 | 拖动松手定位稳、不抖 |
| **C 排版** | 现 sheet 内容嵌入「字体排版」展开区 | 改字号/主题即时生效 |
| **D 模式** | 阅读模式展开区 | 移动可切滚动；桌面无滚动 |
| **E 亮度** | 主题快选展开区 | 四主题切换 |
| **F 听书** | UI 占位 | 不崩溃、文案清晰 |
| **G（可选）** | 书签入口整理 / 目录半屏化 | 另开 |

**建议开工顺序：A → B → C → D → E → F。**

## 5. 非目标（本大改）

- TTS / 听书引擎  
- 笔记、划线、完整「设置」页（电量时间、休眠、词典等）  
- 仿真卷曲真渲染  
- 漫画阅读器同步改版  
- 为追像素完美复刻微信视觉（菱形滑块等可近似，不阻塞功能）

---

## 6. 风险与注意

- Foliate `gap/2` 边距语义已踩过坑；工具条高度变化**不得**改分页 inset（chrome 仍为覆盖层）。  
- Seek 与翻页并发：松手 seek 期间忽略瞬时 relocation 回写，避免滑块跳动。  
- 展开面板高度变化时，正文 WebView 尺寸不变（覆盖层），避免 CFI 漂移。
