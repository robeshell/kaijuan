# 阅读器 Chrome 设计规范

适用范围：漫画阅读器（Phase 1）与后续图书阅读器共享的 chrome 语言。
总纲见 [../DESIGN_FOUNDATION.md](../DESIGN_FOUNDATION.md)，本文件是它的阅读器专项展开。

## 核心原则

- **内容即界面**。漫画页 / 书页占满窗口，chrome 默认隐藏，只在用户召唤时出现。
- chrome 出现时不应让人出戏：玻璃材质、快速淡入淡出、不遮挡阅读焦点区域。
- 所有浮层（面板、菜单、tip、注释编辑器）从同一套面板语言延展，不回退平台默认 Material 表单风格。

## 布局结构

- 阅读区：edge-to-edge，漫画页按模式（slide / spread / static / vertical）排布，背景为中性深灰（不跟随 App 浅色主题——阅读区背景属于"阅读主题"，见下文）。
- 顶栏（召唤时）：左返回、中标题、右动作（书签、设置）。高度 56，玻璃材质。
- 底栏（召唤时）：进度滑条 + 页码 + 模式/方向快捷切换。玻璃材质。
- 召唤方式：单击页面中央区域切换 chrome 显隐；边缘点击翻页；Esc / 系统返回键先关 chrome 再退出阅读器。

## 阅读主题（内容区，独立于 App 主题）

内容区背景不跟随 App 亮/暗切换，而是阅读器内独立选择：

| 主题 | 背景 | 用途 |
|---|---|---|
| 纸白 | #FAFAF8 | 日间 |
| 米色 | #F5F0E6 | 护眼 |
| 深灰 | #1C1C1E | 夜间（漫画默认） |
| 纯黑 | #000000 | OLED / 漫画无边框模式 |

漫画阅读器默认深灰。图书阅读器的文字/强调色在 Phase 3 的图书 spec 中定义。

## Chrome 材质

- 使用 `AppSemantics.glassFill` + `BackdropFilter` 模糊（sigma 20 以内，克制）。
- 深色阅读主题上 chrome 玻璃取深色填充，浅色阅读主题上取浅色填充——chrome 跟随**阅读主题**而非 App 主题。
- 阴影：柔和低位（`BoxShadow` blur 16、y 4、透明度 ≤ 0.12）。
- 圆角：浮层 `AppRadii.panel`（20），控件 `AppRadii.medium`（12）。

## 交互规则

- chrome 显隐动画 200ms 淡入 + 轻微位移（≤ 8dp），不用弹性曲线。
- 进度拖动：拖动中实时预览页码，松手才跳转（一次 seek，不连续跳转）。
- 预加载对 chrome 透明：加载指示只出现在翻页目标未就绪时，常驻 loading 是 bug。
- 桌面端：方向键 / 空格翻页，F 全屏，Esc 退出全屏或关闭 chrome。
- 竖屏紧凑布局下，底栏收敛为进度 + 页码，模式切换收进顶栏菜单。

## 状态显式

- 导入失败、页面解码失败、文件缺失：明确错误页 + 重试/移除入口，不静默跳过。
- 空书库引导导入，空书架引导去书库，不显示裸空列表。
