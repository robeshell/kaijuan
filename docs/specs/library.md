## 目标

**开卷** App 内已导入条目的浏览与管理：漫画（页图）与图书（reflow）混排，通过「类型」筛选。

## 范围

- **做**：网格/列表、导入、打开（按 `kind` 路由）、搜索/排序/筛选（含类型）、多选、**书单**二级、**合集**。  
- **不做**：跳转到另一 App；按品牌身份切库；封面卡上的单独「…」操作入口。

## 信息架构

```text
书库
  · 主区：单本（网格 / 列表）+ 合集卡混排（合集排最前）
  · 过滤器：类型（全部 / 漫画 / 图书）/ 状态 / 上架 / 格式 / 搜索 / 排序
  · 二级入口：
      - 书单 → 长清单管理（lists.md）
      - 合集 → 拼贴盒管理（collections.md）
```

- 导入扩展名：CBZ / ZIP / EPUB。  
- Android SAF 不可靠识别 EPUB MIME；Android 文件选择器允许浏览全部文件，返回应用后再按扩展名严格接收 CBZ / ZIP / EPUB。其他平台继续在选择器阶段过滤扩展名。
- EPUB 按内容自动路由：Dart ZIP/OPF spine 抽样（`EpubKindProbe`）— 正文 → book；抽样中绝大多数章节为低文字页图 → comic（`EpubImportRouter`）；普通封面和零散插图不参与漫画判定。页图路径不经 WebView。
- **书单**与**合集**入口文案、图标必须区分（列表 vs 拼贴盒）。

## 打开路由

```text
点单本 → 按 item.kind 进入对应引擎
  kind=book  → BookReaderScreen
  kind=comic → ComicReaderScreen

点合集卡 → 合集内容页 → 再点成员 → 引擎
点书单 → 书单内容列表 → 再点成员 → 引擎
```

## 卡片 / 空态 / 视觉

- **单本卡**：封面 + 两行标题；可选进度 / 格式角标。无封面上的「…」按钮。  
- **合集卡**：单本卡同尺寸外框 + 内部小封面拼贴；见 collections.md。目录合集卡长按出管理菜单（无角标按钮）。  
- 空态文案：「导入 CBZ、ZIP 或 EPUB」，主按钮「导入」。  
- **顶栏分层**（宽窄同构，禁止桌面单行硬塞搜索+排序+工具）：  
  1. 标题行：书库 / 书单·合集 / 布局切换 / 多选 / 导入（窄屏二级入口为图标）  
  2. 搜索行：通栏搜索（宽屏可 `maxWidth` 约束，不挤在标题行右侧）  
  3. 筛选行：`Wrap` — 排序 + 类型 / 状态 / 上架 / 格式（排序不单独占标题行）  
- Token 与间距见 DESIGN_FOUNDATION。

## 多选

- 批量：删除、上架/移出书架、加入书单、加入合集。  
- **进入多选**：长按单本封面（或顶栏多选）；点选切换；不在封面上挂单独操作钮。  
- 不在多选里混淆「创建书单」与「创建合集」的主按钮文案。
- **封面选中态**：仅角标勾选（实心 `check_circle` / 空心圆）；**不要**封面外加粗强调色描边。多选未选中时角标用白描边圆（带轻阴影）。  
- **封面卡悬停 / 按压**：禁用 Material 泼墨与灰雾 overlay（盖在封面上很脏）；可用透明 ink 或纯 `GestureDetector`。
