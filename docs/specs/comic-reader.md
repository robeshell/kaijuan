# 漫画阅读器设计规范

> 状态与是否做：只改 [../PRODUCT.md](../PRODUCT.md)，此处不维护第二份状态表。

**PRODUCT**：§4.6  
**视觉**：[../DESIGN_FOUNDATION.md](../DESIGN_FOUNDATION.md)  
**相关 spec**：[reader-chrome.md](./reader-chrome.md)

## 目标

页图漫画的沉浸阅读：四模式翻页、方向、主题；补齐页图刚需（缩放、缩略图跳页、亮度）；双页呈对开贴合。

## 范围

- **做**：
  - 四模式（滑动 / 静态 / 纵向 / 双页）+ LTR/RTL + 四主题
  - 进度 locator、书签、偏好持久化
  - **双页贴合**：左右页无缝隙并排，整体 `contain` 居中；窄屏回退单页
  - **页内缩放**：滑动 / 静态 / 双页 pinch；放大时单击复位；不做双击（保单击即时）
  - **缩略图导航**：底栏工具条入口 → 网格跳页
  - **亮度遮罩**：阅读内压暗（非系统亮度），偏好持久化
  - **Chrome**：与图书对齐的不透明顶栏 + 底栏五键工具条
  - **纵向真实宽高比**：按页像素比排版（未知时暂用 0.7）
- **不做**（本能力内）：
  - 文本划线 / 笔记 / 书摘 / 书内 OCR 搜索 / TTS
  - 图书式五键底栏大改
  - 封面单独成页的高级双页配对（仍偶奇配对）
  - 仿真翻页

## 入口与信息架构

- `openReadingItem` → `kind ≠ book` → `ComicReaderScreen`
- 偏好仅在阅读器内调整（`comic_reading.json`）

## 交互

### 翻页与 chrome

1. 滑动 / 静态 / 双页：左右约 25% 翻页，中间显隐 chrome（未放大时）。
2. 纵向：滚动翻页；点按显隐 chrome。
3. **chrome 已显示时任意点按只关 chrome、不翻页**（与图书一致）。
4. 放大（scale > 1）：单击复位缩放；pinch 继续缩放。
5. Esc：先关 chrome / 缩略图，再退出。

### 双页

1. 偶奇配对（0–1, 2–3…）；末页可单页。
2. 两页按像素比例贴合为一条对开，**中间无空隙**；整体等比缩放入视口居中。
3. 视口过窄（宽高比不足以容纳对开）时回退单页（显示当前页）。
4. RTL：视觉左右对调，语义翻页不变。

### 缩放

1. 双指 pinch（滑动 / 静态 / 双页）。
2. **不做双击缩放**（避免与单击显隐 chrome 抢手势、产生 ~300ms 延迟）。
3. 翻页或切模式时重置缩放。
4. 纵向模式不做页内缩放。

### 缩略图

1. 底栏工具条「缩略图」键；网格跳页。
2. 点缩略图 → `jumpTo` 并关闭 sheet。
3. 懒加载解码；与正文共用 `ComicPageCache`。

### 亮度 / 方向 / 模式

1. 底栏五键：缩略图 / 书签 / 亮度 / 方向 / 阅读模式；点键上方展开面板（对齐图书工具条）。
2. 亮度 0.15–1.0；默认 1.0；黑色遮罩不拦截手势。
3. 阅读模式面板含四模式 + 四主题。

## 状态与文案

- 打开失败：错误文案 +「返回」
- 缩略图解码失败：占位破图图标
- 书签：「第 N 页」

## 视觉

- 与图书一致：不透明顶底栏（`GlassBar(blur: false)`）+ 底栏工具条；见 [reader-chrome.md](./reader-chrome.md)
- 双页对开居中；页缝无装饰线（v1）
- 亮度遮罩盖在页图层之上、chrome 之下

## 数据与路由

- Locator：`ComicLocator(pageIndex, pageOrderVersion)`
- 偏好：`mode` / `direction` / `readingTheme` / `brightness`
- Controller：`ComicReaderController`；引擎：`ComicSession` + `ComicPageCache`
